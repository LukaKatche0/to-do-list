შევქმნათ ახალი კომპონენტი app-custom-input, რომელსაც @input-ით გადაეცემა ფროფერთი disabled, თუ true არის ეს ფროფერთი, html-ის input იყოს disabled. გამოვიყენოთ ეს app-custom-input კოპმონენტი app-component-ში და იქიდან გადავცეთ disabled. ექსტრა: შევქმნათ button app component-ში, რომლის დაჭერაზეც disabled-ს შეეცვლება მნიშვნელობა.
